Authorization API

Routes:

  • /auth/login (POST)

    In this path you can authorize yourself by providing the following parameters in the body of the request:
    • username
    • password
    • grant_type: Must be "password"

    Output: 
    
  {
    "success: true,
    "tokens": {
        "accessToken": "...",
        "refreshToken": "...",
        "expiration": "..."
    },
    "user": {
        "username": "...",
        "email": "...",
        "date": "...",
        "lastConnection": "...",
        "name": { 
          "firstname": "...",
          "lastname": "..."
        },
        "roles": [...]
    }
  }

  • /auth/register (POST)

    In this path you can register by providing the following parameters in the body of the request:
    • username
    • password
    • grant_type: Must be "client_credentials"
    • email (optional)
    • firstname (optional)
    • lastname (optional)

    Output: 
    { success: true }

  • /auth/refresh (POST)

    In this path you can refresh your token by providing the following parameters in the body of the request:
    • refreshToken
    • grant_type: Must be "refresh_token"

    Output: 
    
{ 
  "success": true,
  "token": {
    "accessToken": "...",
    "refreshToken": "...",
    "expiration": "..."
  }
}
